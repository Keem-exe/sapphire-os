# Sapphire OS V1

Intelligent Learning Platform for CSEC Mathematics & English

## Overview

Sapphire OS is a gamified learning platform designed to help students master CSEC Mathematics and English through intelligent coaching, interactive lessons, and community engagement.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```
